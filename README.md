# ⚡ AlgoPulse

An interactive, full-stack algorithm visualizer built with **Java (Spring Boot)** and **Vanilla JavaScript**. AlgoPulse breathes life into abstract computer science concepts by translating backend sorting operations into sleek, real-time frontend animations.

---

##  Features
* **4 Core Algorithms:** Watch **Bubble Sort**, **Selection Sort**, **Insertion Sort**, and **Merge Sort** work in real-time.
* **Granular Frame Tracking:** The custom Java backend captures distinct `COMPARE` and `SWAP` events to provide deep step-by-step visibility.
* **Modern UI/UX:** Features a glass-morphic dark theme, intuitive layout cards, and fluid color indicators for changing arrays.
* **Responsive Stage Layout:** Dynamically adjusts element widths to visualize user-provided array snapshots beautifully.

---

##  Tech Stack & Architecture
* **Backend Engine:** Java 21 / 23, Spring Boot (Spring Web), Maven.
* **Frontend Layer:** HTML5, CSS3 (Modern Variables & Gradients), JavaScript (ES6+ Async/Await Engine).
